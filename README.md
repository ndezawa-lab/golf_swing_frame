# Golf Swing Sheet Maker

## アプリの利用目的

ゴルフ練習場でスマホ撮影したスイング動画から、ChatGPT解析に渡しやすいスイング連続写真の一覧画像を作成するためのPWAです。

動画読み込み後にアドレス〜フィニッシュの範囲を自動推定し、40コマ版と16コマ版の一覧画像を同時生成します。生成した画像はスマホから保存・共有して、ChatGPTプロジェクトへアップロードすることを想定しています。
