# Golf Swing Sheet Maker PWA 認証付き版 v9

ゴルフスイング動画から、ChatGPT解析向けの **40コマ一覧画像** を作成するスマホ向けPWAです。  
この版では、公開URLにアクセスしたときに最初にパスワード画面を表示します。

## デフォルトパスワード

```text
swing2026
```

**公開前に必ず変更してください。**

## 追加内容

- パスワード入力画面
- ログイン後にアプリ表示
- セッション中のみログイン状態を保持
- ログアウトボタン
- PWA対応
- ホーム画面追加対応
- オフライン起動対応
- アドレス〜フィニッシュ自動推定
- 40コマ / 5列×8行 / 時刻ラベル付き一覧画像

## パスワード変更方法

### 方法1: 同梱のハッシュ生成ページを使う

1. `password-hash.html` をブラウザで開く
2. 新しいパスワードを入力
3. ハッシュを生成
4. `config.js` を開く
5. `PASSWORD_HASH` の値を生成結果に貼り替える
6. 保存して公開環境へアップロード

`config.js` の該当箇所:

```js
window.AUTH_CONFIG = {
  PASSWORD_HASH: "ここを貼り替える",
  SESSION_KEY: "golf_swing_sheet_auth_v1",
  APP_NAME: "Golf Swing Sheet Maker"
};
```

## 公開方法

GitHub Pages / Cloudflare Pages / Netlify / Vercel などの静的ホスティングに配置できます。  
PWAとして使うにはHTTPS環境が必要です。GitHub Pagesなどを使えばHTTPSになります。

## iPhoneでホーム画面に追加

1. Safariで公開URLを開く
2. 共有ボタンをタップ
3. 「ホーム画面に追加」
4. ホーム画面のアイコンから起動

## Androidでホーム画面に追加

1. Chromeで公開URLを開く
2. メニューをタップ
3. 「アプリをインストール」または「ホーム画面に追加」
4. ホーム画面のアイコンから起動

## 重要な注意

この認証は、静的WEBアプリ上の **簡易パスワードゲート** です。  
URLを知っている人の気軽なアクセスを防ぐ用途には使えますが、業務システムレベルの強固な認証ではありません。

より強い認証が必要な場合は、以下を使ってください。

- Cloudflare Access
- Basic認証付きのサーバー
- Firebase Authentication
- Supabase Auth
- 独自ログイン機能付きのバックエンド

ただし、今回のアプリは動画をサーバーへアップロードしない設計です。  
第三者がURLにアクセスできても、あなたのスマホ内の動画や生成画像が見えるわけではありません。


## v7 変更点

スマホで動画読み込み時にカメラが直接起動してしまう問題に対応しました。

- 「撮影済み動画を選択」ボタンを追加
- 「カメラで新規撮影」ボタンを追加
- 撮影済み動画選択では `capture` を使わない
- カメラ撮影では `capture="environment"` を使う

これにより、スマホ上でファイル選択とカメラ撮影を選び分けられます。


## v8 変更点

スマホPWA環境で「撮影済み動画を選択」「カメラで新規撮影」をタップしても反応しない問題に対応しました。

原因:
- 非表示の `<input type="file">` をJavaScriptの `.click()` で開く方式は、iOS Safari / PWA環境でブロックされることがあります。

対応:
- JavaScriptでファイル選択を開く方式を廃止
- `<label for="...">` と `<input type="file">` を直接紐づける方式に変更
- ファイル入力は `display:none` ではなく、視覚的に隠す方式に変更

これにより、スマホでのタップ反応が安定しやすくなります。


## v9 変更点

iPhone / iOS PWAで生成画像が直接ダウンロードできない問題に対応しました。

対応内容:
- `a.download` だけに依存しない保存方式へ変更
- Web Share API対応端末では共有シートを表示
- 共有できない場合は長押し保存用の画像プレビューを表示
- 「画像だけを開く」ボタンを追加
- Service Workerのキャッシュ名をv9に更新

iPhoneでの保存方法:
1. 「画像を保存 / 共有」を押す
2. 共有シートが出た場合は「画像を保存」または「ファイルに保存」を選択
3. 共有シートが出ない場合は、表示されたプレビュー画像を長押しして保存
